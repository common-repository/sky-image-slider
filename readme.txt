== Description ==
Lite weight, simple and easy to use image slider with caption and numerous transition effects.

= How to use  =
- After install you can see a sidebar menu in the dashboare called "Slider"
- Add one or more Slider from there. 
- You will get Shortcode for every Slider In the slider list.
- Copy Shortcode for slider you wanna publish 
- Past the shortcode in post, page, widget areas To publish them. if you want to publish a slider in template file use <?php echo do_shortcode('SLIDER_SHORTCODE') ?>
- Enjoy !

* For installation help click on Installation Tab

== Installation ==
1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Create slider by clicking on left sidebar menu item called "Slider"
4. Use shortcode in page, post or in widgets.
5. If you want video in your theme php file, Place `<?php echo do_shortcode('SLIDER_SHORTCODE'); ?>` in your templates
